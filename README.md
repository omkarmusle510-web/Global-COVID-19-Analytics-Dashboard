# 🌍 Global COVID-19 Analytics Dashboard

## 📌 Overview

The Global COVID-19 Analytics Dashboard is an interactive data analytics project built using Python, Streamlit, Pandas, and Plotly.

This dashboard provides real-time COVID-19 insights from countries around the world using publicly available COVID-19 datasets. Users can explore key statistics, compare countries, analyze vaccination data, and visualize trends through interactive charts.

---

## 🚀 Features

### 📊 Dashboard Analytics

* Total COVID-19 Cases
* Total COVID-19 Deaths
* Population Statistics
* Vaccination Statistics
* Death Rate Calculation
* Global Country Ranking

### 📈 Data Visualizations

* Top 10 Countries by Total Cases
* Top 10 Countries by Vaccinations
* Population vs Total Cases Scatter Plot
* Interactive Plotly Charts

### 📥 Export Options

* Download full dataset
* Download country-specific data

### 🎨 User Interface

* Interactive Streamlit Dashboard
* Responsive Layout
* Analytics-focused Design
* Sidebar Filters
